ИНФОРМАЦИЯ ДЛЯ ПОНИМАНИЯ, КАК РАБОТАТЬ С ПРОГРАММОЙ

- суть программы в написании 11 функций сотировок и их анализа

1. ПАПКА graphics
	содержит папки, куда сохраняются графики с графиками

2. ПАПКА pythonProgram
	2.1. program.py
		программа для построения графиков для каждого алгоритма, в которую вставляется
		код из соответствующего файла в папке CodesForGraphics (место, куда вставлять код
		определено в программе с помощью комментария). 
	2.2. delete_cash.py
		программа для удаления указанных в ней файлов (картинок с графиками)
		(использовалась, когда сохранялись пустые картинки из-за постоянного вызова функции savefig())

3. ПАПКА results.txt
	содержит папки, в которую сохраняется время работы проргаммы для каждого алгоритма
	каждая строка в файле содержит время работы программы для соответствующего размера массива
	с возрастанием строк возрастает размер массива

4. ПАПКА SortFunc
	Содержит папки с .cpp и .h файлами

4.1. ОБЩИЕ ПОЛОЖЕНИЯ ПО ПРОГРАММЕ
	программа содержит две main() функции, описанные в файлах Program.cpp и TimeMeasuring.cpp
	для запуска программы один из файлов нужно закомментировать

	Файл TimeMeasuring.cpp зависит от файла constants.h, для понимания работы следует прочитать
	комментарии в этих двух файлах, а также отчёт по программе.

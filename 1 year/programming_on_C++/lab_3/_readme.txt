Лабораторная работа №3
Задание:
    написать программу, вычисляющую количество итераций, после которых
    значение затухающей функции будет находиться в отрезке [-eps, eps].
    (eps <= 1e-10)
    функция: f(x) = f(x-1) * x(i+2) / (i+1)^2, где i - шаг итерации.
    программа должна осуществлять форматированный табличный вывод
    всех функции на каждом шаге (каждой итеарции)

Практический навык работы:
    проверка равенства значений дробных переменных (с точностью до eps)
    написание форматированного вывода

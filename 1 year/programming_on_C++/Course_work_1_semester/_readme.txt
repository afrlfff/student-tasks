ЗАДАНИЕ НА КУСРОВУЮ РАБОТУ

ВХОДНЫЕ ДАННЫЕ
points.txt - задает все точки на плоскости построчно

    2 2
    -2 2
    0 4
    0 2

ВЫХОДНЫЕ ДАННЫЕ
protokol.txt - содержит подробное описание работы программы
result.txt - содержит результаты работы программы

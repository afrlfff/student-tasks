Задание вроде бы заключается в том, чтобы распорядиться деньгами банка так, чтобы
    выдать кредиты как можно большему числу клиентов.

    при этом все строки (имена, адреса, названия и т.п) нужно представлять в виде
    блоков фиксированной длины, каждый из которых является массивом символов.
    Пример:
        Иванов И.И. -> |Ива|нов| И.|И.|  -> |И|в|а|н|о|в|И|.|И|.|
    Реализовать это представление с помощью классов и структур.
